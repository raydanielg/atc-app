## üöÄ Getting Started

1. **Clone the repository:**
   ```sh
   git clone <your-repo-url>
   cd <project-directory>
   ```
2. **Install dependencies:**
   ```sh
   yarn install
   # or
   npm install
   ```
3. **Start the app:**
   ```sh
   yarn dev
   # or
   npm run dev
   ```
4. **Run on Android/iOS:**
   ```sh
   yarn android
   yarn ios
   ```
